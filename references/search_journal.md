# 搜索期刊

## 功能说明
在 LetPub 中搜索期刊，支持自动补全搜索和高级多条件筛选。

## 使用方法

### 1. 自动补全搜索

```python
from letpub_skills import search_journal

results = search_journal("CHEMICAL ENGINEERING SCIENCE")
# 返回: [{"id": "1642", "label": "CHEMICAL ENGINEERING SCIENCE", "value": "...", "issn": "0009-2509"}]
```

### 2. 高级搜索

```python
from letpub_skills import search_journals_advanced, parse_search_results, CATEGORY_CN, SUBCATEGORY_CN

# 查看可用的大类学科
print(CATEGORY_CN)  # {0: "医学", 1: "生物学", 2: "农林科学", ...}

# 按学科分类搜索
html = search_journals_advanced(
    searchcategory1="计算机科学",  # 大类学科
    searchcategory2="计算机：人工智能",  # 小类学科（可选）
    searchsort="relevance"
)

# 解析搜索结果
result = parse_search_results(html)
print(f"共 {result['total_records']} 条记录，{result['total_pages']} 页")
for journal in result['journals']:
    print(f"{journal['name']} - IF: {journal['impact_factor']} - {journal['partition']}")
```

### 3. 完整搜索参数

```python
html = search_journals_advanced(
    searchname="",           # 期刊名
    searchissn="",           # ISSN
    searchfield="",          # 研究方向
    searchimpactlow="",      # 影响因子下限
    searchimpacthigh="",     # 影响因子上限
    searchscitype="",        # SCI收录类型: SCIE, SSCI, ESCI, AHCI, SCOPUS, DOAJ
    searchcategory1="",      # 大类学科（中文名称）
    searchcategory2="",      # 小类学科（中文名称）
    searchjcrkind="",        # 新锐期刊分区表: 1, 2, 3, 4
    searchopenaccess="",     # 是否OA: YES, Hybrid, NO
    searchsort="relevance"   # 排序: relevance, impactor, rankscore, jciscore, clickcount
)
```

## 搜索结果字段

| 字段 | 说明 |
|------|------|
| issn | ISSN |
| name | 期刊名称 |
| journal_id | 期刊ID |
| shortname | 期刊简称 |
| impact_factor | 影响因子 |
| real_time_if | 实时IF |
| h_index | h-index |
| cite_score | CiteScore |
| partition | 分区（如"1区"） |
| field | 学科分类 |
| sci_type | SCI收录类型 |
